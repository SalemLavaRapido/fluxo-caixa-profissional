Configurar Conexão com Supabase

Para usar este sistema na nuvem, você precisa configurar o Supabase:

  2. Crie um novo projeto
  3. Vá em Settings > API
  4. Copie a URL e a API Key
  5. Execute o SQL do arquivo setup.sql
  6. Clique no botão "Configurar" abaixo
